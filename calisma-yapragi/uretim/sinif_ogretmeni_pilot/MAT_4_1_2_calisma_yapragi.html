1. (1) 2, (2) değer, (3) altmış bin
1) 452.301'de binlikler basamağındaki rakam 2'dir. 2) Her basamağın ayrı bir değeri vardır. 3) On binler basamağındaki 6 → 6×10.000 = 60.000 (altmış bin) eder.
2. A
İki yüz otuz beş bin, dört yüz bir → 235.000 + 401 = 235.401. Doğru cevap A seçeneğidir.
3. {"r0c1": "5", "r0c2": "2", "r1c1": "0", "r1c2": "8", "r2c1": "7", "r2c2": "0"}
472.156: onlar basamağı 5, binler basamağı 2. 308.602: onlar basamağı 0, binler basamağı 8. 150.073: onlar basamağı 7, binler basamağı 0.
4. 1-D, 2-D, 3-Y
1. Doğru: 520.304 = 500.000 + 20.000 + 300 + 4. 2. Doğru: 306.201 = 300.000 + 6.000 + 200 + 1. 3. Yanlış: 0 değerli basamaklar da genişletilmiş modelde gösterilir.
5. A
Her iki sayının da yüz binleri ve on binleri aynıdır (425.0__). Yüzler basamağında 1 > 0 olduğundan 425.130 > 425.031'dir.
6. 512.003 → 512.030 → 520.130 → 521.003
512.003 < 512.030 < 520.130 < 521.003. Yüz binler basamağında hepsi 5'tir. On binler basamağında 1 ve 2 vardır; 1 < 2 olduğu için 512.0xx'ler küçüktür. Binler basamağında 2 > 0 olduğundan 512.003 < 512.030.
7. 305.402 = 300.000 + 0 + 5.000 + 400 + 0 + 2 = 300.000 + 5.000 + 400 + 2
3×100.000 = 300.000, 0×10.000 = 0, 5×1.000 = 5.000, 4×100 = 400, 0×10 = 0, 2×1 = 2. Toplam: 300.000 + 5.000 + 400 + 2 = 305.402.
8. A
250.000 − 138.500 = 111.500. Doğru cevap A seçeneğidir.
