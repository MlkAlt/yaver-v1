MAT.4.1.3Sayıları Sıralayalım
Öğretmen içindir —
dağıtmadan önce ayırınız.
1. C
Yüz binler basamağında hepsi 1'tir. On binler basamağında 2 ve 0 vardır; 0 < 2 olduğu için 120.500 en küçüğüdür.
2. 1-Y, 2-D, 3-D, 4-D
1. Yanlış: 320.150 < 321.050. 2. Doğru: En soldaki basamak en yüksek değerdedir. 3. Doğru: 400.000 > 399.999. 4. Doğru: Karşılaştırmaya soldan başlanır.
3. 451.023 → 415.320 → 415.230 → 405.230
451.023 > 415.320 > 415.230 > 405.230. Yüz binler basamağında hepsi 4. On binler basamağında 5 > 1 > 0.
4. 1-A, 2-B, 3-C
1 → A: 203.100 < 230.010 < 230.100 (binler basamağında 3 > 0). 2 → B: 150.200 > 150.020 > 150.002 (yüzler basamağında 2 > 0). 3 → C: 300.003 < 300.030 < 300.300.
5. D
520.010 < 520.015 < 520.020. 520.015, 520.010 ile 520.020 arasındadır.
6. Sıralama: Ali (180) < Can (205) < Ayşe (230) < Zeynep (250) < Mehmet (305). Fark: 305 − 180 = 125 sayfa.
Küçükten büyüğe: Ali (180), Can (205), Ayşe (230), Zeynep (250), Mehmet (305). Fark: 305 − 180 = 125.
7. C
C seçeneğinde 420.003, 420.300'den küçük olmalıdır: 420.003 < 420.030 < 420.300. Ancak verilen sıralamada 420.300 < 420.003 olarak gösterilmiştir ki bu yanlıştır.
8. 1-D, 2-D, 3-Y
1. Doğru: 500.000 > 499.999. 2. Doğru: 123.456 < 123.465. 3. Yanlış: 300.000 > 299.999.
