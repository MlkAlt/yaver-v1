1) İleriye doğru sayarken sayılar büyür. 2) Geriye doğru sayarken sayılar küçülür. 3) 3'er 3'er: 108, 111, 114.
2. C
5'er 5'er saymada: 25, 30, 35, 40, 45, 50... 42 bu sıralamaya uymaz çünkü 42, 5'in katı değildir.
3. 1-D, 2-D, 3-D
1. Doğru: 100'den 4'er 4'er geriye: 100, 96, 92, 88, 84. 2. Doğru: 200 (1), 210 (2), 220 (3), 230 (4), 240 (5). 3. Doğru: 50, 47, 44, 41 — 4. sayı 41'dir.
4. 10'er 10'er sayma → 2'er 2'er sayma → 3'er 3'er sayma → 7'şer 7'şer sayma
En basit: 10'er 10'er (onluk sisteme uygun). Sonra 2'er 2'er (çiftler). Sonra 3'er 3'er. En karmaşık: 7'şer 7'şer.
5. B
150 (1), 175 (2), 200 (3), 225 (4). 4. sayı 225'tir.
6. a) 50, 57, 64, 71, 78. 5. sayı 78'dir. b) 50 + 7n > 200 → 7n > 150 → n > 21,4. 22. adımda: 50 + 7×22 = 204. 200'ü 22. adımda aştılar. c) 200, 193, 186, 179. 4. sayı 179'dur.
a) 50 → 57 → 64 → 71 → 78 (5. sayı). b) 50 + 7n > 200 → n = 22 (204). c) 200 → 193 → 186 → 179 (4. sayı).
